## Build

1. Install [Node.js](https://nodejs.org/en/) >= 4

2. Prepare git submodule
    ```
    git submodule init
    git submodule update
    ```

2. Run `npm install` to install dependendencies

3. Run `npm test` to build the `games` directory and [`Lutro.dat`](Lutro.dat)
